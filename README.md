# GenAI-Powered Customer Support Quality Auditor

## Milestone 1 & 2

This repository contains:

- Audio transcription using Whisper
- Chat normalization
- LLM-based scoring engine for empathy, compliance, and resolution

This is the project structure:

data/ → raw chats and calls  
src/transcription → speech-to-text module  
src/scoring → LLM evaluation engine  
src/utils → preprocessing utilities  
main.py → pipeline integration
